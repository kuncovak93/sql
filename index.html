SQL Injection

1. Úvod

SQL Injection je jedna z najnebezpečnejších zraniteľností webových aplikácií. Vzniká, keď aplikácia nesprávne spracuje vstup od používateľa.

2. Čo je SQL Injection

  • obídenie prihlasovania
  • zobrazenie citlivých údajov
  • mazanie dát

3. Zraniteľný kód

SELECT * FROM users 
WHERE username = '$username' AND password = '$password';

4. Ukážka útoku

Zadaj do formulára:

' OR '1'='1
