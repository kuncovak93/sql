SQL Injection

1. Úvod

SQL Injection je jedna z najnebezpečnejších zraniteľností webových aplikácií. Vzniká, keď aplikácia nesprávne spracuje vstup od používateľa.

2. Čo je SQL Injection

  • obídenie prihlasovania
  • zobrazenie citlivých údajov
  • mazanie dát
  • úplné ovládnutie databázy

3. Zraniteľný kód

SELECT * FROM users 
WHERE username = '$username' AND password = '$password';

4. Ukážka útoku

Skús zadať do mena alebo hesla:

' OR '1'='1

Simulácia prihlasovania

5. Následky útoku

  • únik databázových údajov
  • zobrazenie emailov a hesiel
  • mazanie alebo úprava dát
  • možné ovládnutie systému

6. Ako sa chrániť

  • používanie prepared statements
  • kontrola vstupov
  • používanie bezpečných frameworkov
  • testovanie bezpečnosti

7. Záver

SQL Injection je vážna bezpečnostná chyba, ktorá môže spôsobiť únik citlivých údajov. Preto je dôležité myslieť na bezpečnosť už pri vývoji aplikácií.
